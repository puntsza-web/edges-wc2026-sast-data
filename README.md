# edges-wc2026-sast-data

Open World Cup 2026 fixture data in South African Standard Time (SAST).

## Dataset

This dataset provides the complete FIFA World Cup 2026 match fixture schedule, with all kick-off times converted to South African Standard Time (SAST, UTC+2). It is intended for developers, researchers, and fans who want a clean, open reference for fixtures localized to SAST.

## Source and Files

- Project page: https://www.edges.co.za/data/wc2026-fixtures/
- CSV: https://www.edges.co.za/data/wc2026-fixtures.csv
- JSON: https://www.edges.co.za/data/wc2026-fixtures.json

## Scope

No odds, no picks, no betting predictions. This is fixture data only.
